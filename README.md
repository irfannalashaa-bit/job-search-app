# Job Search App

A web-based job search application built with **n8n** and the **JSearch API**.

## Features

- Search jobs by title
- Filter by city and country
- Remote job support
- Employment type filters
- Date posted filters
- Paginated results
- Export results to Excel (.xlsx)

## Tech Stack

- n8n
- JSearch API (RapidAPI)
- HTML/CSS/JavaScript
- Render

## Workflow

The repository contains the n8n workflow used to power the application.

## Deployment

The application is intended to be deployed on Render using n8n.
